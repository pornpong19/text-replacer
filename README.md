# Text Replacer

แอพพลิเคชั่นเดสท็อปสำหรับแก้ไขคำในไฟล์ข้อความ ใช้ Electron UI และ Python backend

## คุณสมบัติ

- รองรับการเลือกไฟล์ `.txt` หลายไฟล์พร้อมกัน
- สร้างกฎการแทนที่คำได้ไม่จำกัด
- UI สไตล์มินิมอลสีขาว-เทา-ดำ
- รองรับภาษาไทยและ UTF-8
- แสดงผลลัพธ์การดำเนินการแบบละเอียด

## การติดตั้ง

### ความต้องการของระบบ

- Node.js (version 16 หรือใหม่กว่า)
- Python 3.6 หรือใหม่กว่า
- npm หรือ yarn

### ขั้นตอนการติดตั้ง

1. คัดลอกโปรเจค:
```bash
git clone <repository-url>
cd text-replacer
```

2. ติดตั้ง dependencies:
```bash
npm install
```

3. รันแอพพลิเคชั่น:
```bash
npm start
```

### การรันในโหมด Development

```bash
npm run dev
```

### การสร้าง Executable

```bash
npm run build
```

## การใช้งาน

1. **เลือกไฟล์**: คลิกปุ่ม "เลือกไฟล์ .txt" เพื่อเลือกไฟล์ที่ต้องการแก้ไข
2. **ตั้งค่ากฎการแทนที่**: ใส่คำเดิมและคำใหม่ที่ต้องการแทนที่
3. **เพิ่มกฎใหม่**: คลิก "เพิ่มกฎใหม่" หากต้องการแทนที่หลายคำ
4. **ดำเนินการ**: คลิก "ดำเนินการแทนที่คำ" เพื่อเริ่มการทำงาน
5. **ดูผลลัพธ์**: ตรวจสอบผลลัพธ์ในส่วนล่างของแอพ

## ตัวอย่างการใช้งาน

กฎการแทนที่:
- `เบาๆ` → `เบาเบา`
- `เร็วๆ` → `เร็วเร็ว`
- `ใหญ่ๆ` → `ใหญ่ใหญ่`

## โครงสร้างโปรเจค

```
text-replacer/
├── src/
│   ├── main.js          # Electron main process
│   ├── preload.js       # Preload script
│   ├── index.html       # UI หลัก
│   ├── style.css        # Styles
│   └── renderer.js      # Frontend logic
├── backend/
│   └── text_processor.py # Python backend
└── package.json
```

## การแก้ไขปัญหา

### ปัญหาที่อาจพบ

1. **Python script ไม่ทำงาน**: ตรวจสอบว่าติดตั้ง Python 3 แล้ว
2. **ไฟล์ไม่อัพเดต**: ตรวจสอบสิทธิ์การเขียนไฟล์
3. **ตัวอักษรผิดเพี้ยน**: ตรวจสอบ encoding ของไฟล์ต้นฉบับ

### การ Debug

รัน Electron ในโหมด development เพื่อดู console logs:
```bash
npm run dev
```
